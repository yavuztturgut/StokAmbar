# Stok Takip Sistemi

Basit ve kullanışlı bir stok yönetim uygulaması. Ürün ekleyebilir, çıkartabilir ve tüm işlemleri takip edebilirsiniz.

## Kurulum

Gerekli paketleri yükle:
```bash
npm install
```

## Geliştirme Modu

Uygulamayı çalıştır:
```bash
npm run dev
```

Tarayıcında aç: [http://localhost:3000](http://localhost:3000)

## Özellikler

- ✅ Ürün ekleme ve silme
- ✅ Stok artırma/azaltma
- ✅ İşlem geçmişi
- ✅ Veritabanı desteği (Prisma)

## Teknolojiler

- **Next.js** - Web framework
- **React** - Kullanıcı arayüzü
- **Prisma** - Veritabanı yönetimi
- **Tailwind CSS** - Stil
- **TypeScript** - Tip güvenliği

## Komutlar

- `npm run dev` - Geliştirme sunucusu
- `npm run build` - Yapıya dönüştür
- `npm start` - Yapıyı çalıştır
- `npm run lint` - Kod kontrolü
