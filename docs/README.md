# Docs

Bu klasor proje dokumantasyonlari icin ayrildi.

Onerilen kullanim:

- `architecture.md`: uygulama mimarisi ve veri akisi
- `api.md`: endpoint ozeti ve request/response notlari
- `deployment.md`: production kurulum ve ortam degiskenleri
- `decisions.md`: alinan teknik kararlar ve gerekceleri

Kural:

- Kok dizinde yalnizca `README.md` ana giris dokumani kalir.
- Ayrintili veya zamanla buyuyecek belgeler `docs/` altina konur.
